GIT VÀ GITHUB.COM

I. Kiểm tra và cấu hình

Bước 1. Cài đặt git, đăng ký tài khoản github.com

Bước 2. Kiểm tra version

Mở cmd.exe
Gõ: git --version

Bước 3. Cấu hình git trên máy tính

Mở cmd.exe
Gõ: git config --global user.name "HoDienLoi"
Gõ: git config --global user.email "hodienloi@gmail.com"
Gõ: git config --list kiểm tra
Xoá tên: git config --global --unset user.name
Xoá email: git config --global --unset user.email

II. Cấu hình SSH Key

Link thao khảo

https://docs.github.com/en/authentication/connecting-to-github-with-ssh/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent?platform=windows

https://www.youtube.com/watch?v=a-zX_qc2S-M

Thực hiện:

Mở cmd.exe và chạy các lệnh sau
Bước 1. ssh-keygen -t ed25519 -C "hodienloi@gmail.com"
Bước 2. Mở file tài C:\Users.ssh\id_ed25519.pub Sao chép nội dung trong tập tin
Bước 3. Mở website https://github.com/settings/keys New SSH key Giá nội dung đã sao chép ở bước 2

III. Một số lệnh cơ bản

1. Xem trạng thái và xem cấu hình

    git status
    git config --list

2. Cấu hình git

    git init

3. Thêm tập tin vào git

Thêm

    git add file1 file2 folder1 folder2
    git add .
    git add -A
    git add --a

Xoá

    git rm --cached filename
    Di chuyển file
    git mv

4. Chuyển file từ trạng thái workspace, staging sang local repository

Chuyển từ staging sang local repository

    git commit -m "ghi nội dung commit"

Chuyển từ workspace sang local repository

    git commit -a "ghi nội dung commit"

5. Chọn nhánh làm việc

Tạo nhánh mới

    git branch TenBuoi

Chuyển sang nhánh mới

    git checkout TenBuoi

Tạo nhánh và chuyển sang nhánh mới

    git checkout -b TenBuoi

6. Liên kết local repository(PC) với remote repository(Github)

Trên github.com tạo remote repository(Github)

Liên kết local repository(PC) với remote repository(Github)

    git remote add origin git@github.com:hodienloi1981/cdudw1.ccq2211g.git

Nếu git remote sai chúng ta có thể xóa, say đó thiết lập lại

    git remote remove origin

7. Đưa nội dung từ local respository lên remote respository

    git push -u origin TenNhanh

IV. Đẩy project từ local repository(PC) lên remote repository(Github)

1. Lần đầu

Mở cmd.exe và thực hiện các lệnh

    Bước 1. git init
    Bước 2. git add .
    Bước 3. git commit -m "ghi nội dung commit"
    Bước 4. git branch -M master
    Bước 5. git remote add origin git@github.com:hodienloi1981/cdudw1.ccq2211g.git
    Bước 6. git push -u origin master

Chú ý

Bước 4. Mặc định là nhánh master nên khỏi cần

2. Đưa bài đề nhánh master đã tồn tại

Mở cmd.exe và thực hiện các lệnh

    Bước 1. git add filename
    Bước 2. git commit -m "ghi nội dung commit"
    Bước 3. git branch -M master
    Bước 4. git remote add origin git@github.com:hodienloi1981/cdudw1.ccq2211g.git
    Bước 5. git push -u origin master

Chú ý

Bước 4. Đã có rồi thì không cần, kiểm tra bằng lệnh git config --list

3. Đưa bài lên nhánh mới của remote respository đã tồn tại trên github

Mở cmd.exe và thực hiện các lệnh

    Bước 1. git init
    Bước 2. git add filename
    Bước 3. git commit -m "ghi nội dung commit"
    Bước 4. git checkout -b TenBuoi
    Bước 5. git remote add origin git@github.com:hodienloi1981/cdudw1.ccq2211g.git
    Bước 6. git push -u origin TenBuoi

Chú ý

Bước 4. Đã có rồi thì không cần, kiểm tra bằng lệnh git config --list
